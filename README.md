## SEO Features

- Unique `<title>` and `<meta description>` on every page
- JSON-LD structured data (LocalBusiness, Service, FAQPage, BreadcrumbList)
- Open Graph + Twitter Card tags
- Canonical URLs + hreflang (pl/uk)
- Semantic HTML with ARIA labels
- Sitemap.xml + robots.txt
- PL/UA language toggle (client-side)
- FAQ accordion (rich snippet eligible)
- Mobile-first responsive design

## Deployment

Designed for deployment at `https://kurylenko.pl`.
Can be hosted on GitHub Pages, Netlify, Vercel, or any static host.

For GitHub Pages with custom domain:
1. Add `CNAME` file with `kurylenko.pl`
2. Configure DNS A records to GitHub Pages IPs
3. Enable HTTPS in repository settings

## Tech Stack

Pure HTML + CSS + Vanilla JS. No frameworks, no build step needed.
Fonts: Manrope (headings) + Plus Jakarta Sans (body) via Google Fonts.
